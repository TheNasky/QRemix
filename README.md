# QRemix

Sitio web para generar QRs desde textos, links, telefonos, whatsapp, wifi, etc, realizado con React y Sass.

# Dependencias

-  reduxjs/toolkit
-  react-redux
-  formik
-  react-router-dom
-  sass
-  qrcode.react
-  yup
-  semantic-ui-react
-  semantic-ui-css
-  save-svg-as-png
